# NorCal FTC Sponsors

A scraper + static website that collects and displays sponsor information for all teams in the **FIRST Tech Challenge California Northern (USCANO) region**.

Live site auto-updates every Monday via GitHub Actions.

---

## 🚀 Quick Start

### 1. Clone & install
```bash
git clone https://github.com/YOUR_USERNAME/norcal-ftc-sponsors.git
cd norcal-ftc-sponsors
# No npm install needed — scraper uses only Node.js built-ins
```

### 2. Run the scraper
```bash
node scripts/scrape.js
```

This writes three files to `data/`:
- `sponsors.json` — full team + sponsor data
- `sponsors.csv` — same data as a spreadsheet
- `sponsor_frequency.json` — sponsors ranked by how many teams they appear on

### 3. Preview the website locally
```bash
npx serve docs
# or
python3 -m http.server 8080 --directory docs
```

Then open http://localhost:8080

---

## 🛠 Scraper options

```bash
# Different region
node scripts/scrape.js --region USCASO

# Different season year
node scripts/scrape.js --season 2024

# Only scrape top 50 ranked teams (faster)
node scripts/scrape.js --top 50

# Combine flags
node scripts/scrape.js --region USCANO --season 2025 --top 100
```

---

## 🌐 Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **"Deploy from a branch"**, branch `main`, folder `/docs`
4. Your site will be live at `https://YOUR_USERNAME.github.io/norcal-ftc-sponsors/`

The GitHub Actions workflow (`.github/workflows/scrape.yml`) will automatically re-scrape every Monday at 6am UTC and push updated `data/` files, keeping the site current.

You can also trigger it manually from the **Actions** tab.

## ⚙️ How it works

1. **Scraper** hits `ftc-events.firstinspires.org/2025/region/USCANO` to get the full team list
2. Fetches each team's individual page in parallel batches (8 at a time, politely throttled)
3. Parses sponsor names, city, website, rookie year from the HTML
4. Optionally fetches OPR rankings from `ftcstats.org`
5. Writes JSON + CSV output

No API key needed. Data is publicly available on FIRST's website.

## 🔧 Customizing

Want to track a different region or add more data fields? Edit the config block at the top of `scripts/scrape.js`:

```js
const REGION = "USCANO";   // change to any FIRST region code
const SEASON = "2025";     // change to any season year
const CONCURRENCY = 8;     // parallel requests (keep ≤ 10 to be polite)
const DELAY_MS = 200;      // ms delay between batches
```

## 📄 Data notes

- Sponsor data comes directly from teams' FIRST registration — accuracy varies by team
- "Family/Community" entries are filtered out as they're not corporate sponsors
- Teams with no sponsors listed show "no sponsors listed" on the site
- OPR (Offensive Power Rating) comes from ftcstats.org when available
